Особенность примтивных типов данных в java - строго фиксированный размер, независимо от платформы

Примитивные типы данных делятся на:
- Целочисленные
- С плавающей точкой
- Символьные
- Логичские

Целочисленные:

Byte 
- 8 бит
- от -128 до 127
- используется для экономии памяти
- !T.к. все целочисленные литералы по умолчанию int - обязательно явное приведение перменных!

Short
- 16 бит 
- от -32 768 до 32 767
- практически не используется, заменяется на int
- - !T.к. все целочисленные литералы по умолчанию int - обязательно явное приведение перменных!

Int
- 32 бита
- от -2 147 483 648 до 2 147 483 647
- используется повсеместно
- !Стандартный тип для целых чисел в java!

Long
- 64 бита
- от -9 223 372 036 854 775 808 до 9 223 372 036 854 775 807
- нужен, если число не помещается в диапазон int
- !Т.к. все целочисленные литералы по умолчанию int - обязателен суффикс L\l!

С плавающей точкой:

Float
- 32 бита
- 1 бит знака, 8 экспонента, 23 мантисса
- !Т.к. все числа с плавающей точкой по умолчанию double - обязателен суффикс F\f!

Double
- 64 бита
- 1 бит знака, 11 экспонента, 52 мантисса
- !Стандартный тип для чисел с плавающей точкой в джава!

Символьные:

Char

Логические:

Boolean
