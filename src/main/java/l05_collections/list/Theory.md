Iterable -> Collection -> List:
1) ArrayList
2) LinkedList
3) Vector
4) CopyOnWriteArrayList

Контракт Collection добавляет базовые методы
Контракт List добавляет методы для работы с индексами

Контракт List:
1) Элементы имеют целочисленные индексы от 0 до size()-1. 
2) Допускаются дубликаты (два одинаковых объекта могут быть под разными индексами). 
3) Допускается null (если реализация не запрещает явно; ArrayList разрешает). 
4) Гарантируется порядок итерации – от индекса 0 до size-1.

# Arraylist

Фактически это динамический массив с готовыми методами

Контракт ArrayList:
1) RandomAccess – сигнал, что доступ по индексу выполняется за O(1) (в отличие от LinkedList). 
2) Cloneable – поддерживает метод clone() (поверхностное копирование массива). 
3) Serializable – может быть сериализован.

ArrayList подходит для решения 90% задач:
- Доступ к элементам за O(1), итерация по кэшу
- Хранит ссылки в непрервыном массиве
- !Плохо работает для вставки\удаления в середине

Имеет 3 конструктора (ни один из них не выделяет память под обьекты сразу, ленивая инициализация)
Ленивая инициализация
1) ArrayList() - пустой список, после первого добавления начальная емкость станет 10
2) ArrayList(int initialCapacity) - с указанием начальной емкости
3) ArrayList(Collection<? extends E> c) - их другой коллекции, оба списка ссылаются на одни и те же обьекты
!Во всех трех инициализируем по правилу: максимально абстрактный нужный класс - поэтому лучше на уровне List
!Всегда инициализируется с дженериком (даже когда тип неизвестен <T>)

Методы:
1) Добавление  элементов
add(element) - вставить в конец
O(1)
add(index, element) - вставить по индексу
O(n) где n - элементы справа
обычно если места не хватает - список расширяется в 1.5 раза
2) Получение и замена элементов
get(index) - получить по индексу
O(1)
set(index, element) - заменить по индексу
O(1)
3) Удаление элементов
remove(index) - удалить по индексу
Возвращает удаленный элемент
O(n)
remove(element) - удалить по значению
Хорошая практика целочисленные всегда писать через Integer.valueOf()
Возвращает true, если находит
!Удалаяет только первое вхождение
O(n)
Для всех вхождений:
removeAll() или removeIf() - через условие часто чище и эффективнее
