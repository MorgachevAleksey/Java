String - обьект, последовательность символов

Способы создания строк:
1. Строковый литерал
Проверяет - есть ли строка в пуле, возрвращает - если есть, добавляет - если нет
String s1 = "Hello";
2. Оператор new
Даже если есть обьект в пуле, создается новый обьект в куче
(Вообще литерал добавляется в пул - а потом копируется и создается обьект в куче)
String s2 = new String("Hello");
3. Конструкторы
Создает строки из массива символов или байт с помощью встроенных в класс String конструкторов
char[] chars = {'H', 'e', 'l', 'l', 'o'};
String s3 = new String(chars);

Строки в java неизменяемы
Поэтому любая "изменяющая" операция - создает новый обьект String
Нужно для:
- Безопасности
- Кэширования и производительности
- Многопоточности

Литеральные и интернированные строки хранятся в пуле, который в свою очередь находится в куче
Строки созданные new или динамически (считывание файла) - обычне обьекты в куче
intern() - проверяют есть ли строка в пуле, возрвращает - если есть, добавляет - если нет
Интернирование нужно как раз для обьектов в куче

Сравнение строк:
1. equals() - сравнивает содержимое строк
2. == - сравнивают ссылки на обьекты

Конкатенация строк:
1. plus
2. concat()
3. StringJoiner
4. join()

Основные методы класса String:
1. Длина строки
length()
2. Доступ к символам
charAt(index)
3. Поиск подстроки\символа
indexOf()
lastIndexOf()
contains()
startsWith()
endsWith()
4. Извлечение подстроки
substring(int beginIndex, int endIndex)
5. Изменение регистра
toLowerCase()
toUpperCase()
6. Удаление пробелов
trim()
strip()
stripLeading()
stripTrailing()
7. Замена
replace()
replaceAll()
replaceFirst()
8. Разделение строк
split()
